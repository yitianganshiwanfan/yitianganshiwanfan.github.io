工控安全概览

如信息安全概览所总结，从技术领域分，信息安全大体可以分为：

  • Web安全
    • =网络安全
  • 移动端安全
  • 工控安全

此处主要介绍工控领域内的安全：

  • 工控安全，即在工控领域的安全相关技术的统称
    • 工控，全称工业控制
      • 工业，包含很多行业
        • 按照工业4.0
          • 抽象概念
        • 工业控制的细分领域众多
          • 据调查近年来工控安全事件涉及超过 15 个行业
          • 但目前工控市场安全只覆盖到了其中部分行业，要实现全面防护还有许多路要走
    • 工控安全和传统互联网安全关系
    • 工控安全概况
