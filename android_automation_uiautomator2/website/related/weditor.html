weditor

折腾u2期间，少不了要调试设备当前的页面，以及希望了解其中的元素和细节。

这时候，同一个作者开发的，用于辅助u2的weditor，就可以派上用场了。

  • 主页
    • Github
      • openatx/weditor: web editor for atx

安装：

pip3 install -U weditor

启动：

python -m weditor

会自动调用浏览器并打开网址：

http://localhost:17310

效果：

Coding中可以调试代码

加上print后

print(d(className="android.view.View"))

不选中，点击 运行按钮，表示全部运行：

可以实时调试，很方便。

Hierarchy支持有限的搜索
