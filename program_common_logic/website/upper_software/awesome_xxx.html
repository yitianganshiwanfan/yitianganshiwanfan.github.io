对于很多技术，都有热心人去整理很多相关的优秀的资源，供开发者学习和使用，然后叫做awsome xxx

-> 换句话说：如果想要快速找到某个技术的优秀资源汇总，可以去google中搜awesome xxx，往往可以快速方便的找到你要的。
