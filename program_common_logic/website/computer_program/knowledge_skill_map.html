知识技能图谱

知识技能图谱

在学习不同领域的技术时，往往会发现，不同领域所需要学习的内容和方向差距还是很大的，感觉算是隔行如隔山，换个技术领域和方向，很多名词甚至都没听说过。

另外，在某个领域有了深入专研之后，往往可以总结出很多该领域相关的技术和名词，整理出成系统的内容，往往就叫做该领域的技术图谱，知识图谱，技能树，技术成长路线图。
