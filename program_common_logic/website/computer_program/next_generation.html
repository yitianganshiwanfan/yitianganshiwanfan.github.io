Next Generation

Next Generation

有些软件，代码，功能，在开发了新一代，下一代，功能更好更强的版本后，有的会以Next Generation去命令。

比如之前的：

嵌入式的交叉编译的工具crosstool-ng

crosstool 的下一代，新一代版本，叫做：crosstool-ng

用于科学上网的shadowsocks的Mac版本Shadowsocks-NG

之前叫做ShadowsocksX，其新版本，下一代，叫做Shadowsocks-NG
