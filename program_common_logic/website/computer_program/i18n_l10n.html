i18n和l10n

i18n和l10n

在开发软件、APP时，往往会涉及到除了支持中文之外，还要支持其他国家的语言，比如英语、法语、德语等等，专业叫法叫做：多国语言（版本）或者国际化

而在谈论多国语言、国际化时，常常会遇到这个词：i18n

其实i18n==internationalization == 国际化 == 多国语言

-> 之所以这么叫是由于internationalization单词太长，把中间的18个字母internationalizatio省略掉，简写为了i18n

和i18n的类似的含义的类似叫法还有：l10n== localization== 本地化
