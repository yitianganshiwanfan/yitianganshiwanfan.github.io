ideviceinstaller

ideviceinstaller

在Mac中，想要查看（已通过USB连接上的）iOS设备中已安装的应用信息：

  • app名称
  • app包名=bundle id
  • app版本
  • 等

可以用：

  • ideviceinstaller
    • 作用：列出已安装的app信息

安装ideviceinstaller

brew install --HEAD ideviceinstaller

安装后就有了：

  • ideviceinstaller
    • 内部会自动安装额外的依赖
      • libusb
      • libusbmuxd
      • libimobiledevice
        • 其包含多个工具：
          • idevice_id
          • 等
      • libplist
      • libtasn1
      • libzip

使用

语法：

ideviceinstaller -l
