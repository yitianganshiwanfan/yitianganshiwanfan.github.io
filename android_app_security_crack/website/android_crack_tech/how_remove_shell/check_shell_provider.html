如何判断哪家加固方案

都是典型的腾讯乐固的相关内容。

360加固保的加固的目录结构

360加固后的apk经过dex2jar反编译后的目录结构是：

  • com.qihoo.util
  • com.qihoo360.replugin
  • com.stub

这种结构就说明是360加固保加固的。
