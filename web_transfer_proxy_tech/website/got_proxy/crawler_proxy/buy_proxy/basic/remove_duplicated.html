是否去重

是否去重

对于有些业务，可能会要去发出的请求，不能有重复的IP

对此，IP代理提供商，有的确保IP不重复，有的不确保，需要你自己过滤和去重。

比如：

  • 套餐购买 - 芝麻HTTP代理
    • 永久去重
      • 永远不会用到重复的IP

就不需要自己再去去重。
