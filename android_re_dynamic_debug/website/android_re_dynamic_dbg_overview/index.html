Android逆向动态调试概览

Android逆向动态调试概览

关于安卓逆向，之前已通过安卓应用的安全和破解 (crifan.org)进行了一定的介绍。

而安卓逆向期间，除了静态分析去研究反编译的代码外，想要了解安卓app的底层机制和逻辑，需要通过动态调试。

而目前安卓的动态调试的主要的方法是：用Android Studio去调试apk反编译得到的smali代码。

此教程主要就是介绍这方面的详细内容。
