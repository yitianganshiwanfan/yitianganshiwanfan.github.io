Xposed简介

Xposed简介

  • XPosed
    • 是什么：本身是一个框架
      • 所以也称：XPosed框架
        • =Xposed Framework
    • 适用对象：安卓手机
    • 作用：可以在XPosed中安装各种插件，实现各种高级功能
      • 引申：常被用来配合破解安卓应用
    • 前提：需要安卓手机有root权限
    • 特点
      • 开源
      • 支持插件
        • 安装插件后，可以实现各种原本很难实现的效果
