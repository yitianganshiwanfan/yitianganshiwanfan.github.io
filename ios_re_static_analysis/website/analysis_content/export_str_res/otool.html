  • otool
    • =object file displaying tool
    • 是什么：针对目标文件的展示工具
    • 用途：用来发现应用中使用到了哪些系统库，调用了其中哪些方法，使用了库中哪些对象及属性
      • 比如
        • 查看iOS的Mach-O格式的二进制文件的信息
    • 来源：Xcode自带的常用工具
  • 相关
    • 比otool更好的：jtool
    • otool的GUI版：otx
      • x43x61x69/otx: The Mach-O disassembler. Now 64bit and Xcode 6 compatible.
        • https://github.com/x43x61x69/otx
